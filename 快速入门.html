① 一页地图：这篇论文到底在干嘛

先把故事讲成四步。任何时候忘了自己讲到哪，回到这四步。

第 1 步 · 问题：超分是"多解"的

一张低清图 x 对应一整个集合的合法高清解（所有满足 Ay = x 的 y）。 用 MSE 训练 = 取后验均值，把多种纹理平均 → 糊。 本文主张改取后验众数（MAP）：不平均、落在真实图像先验的高概率区。

第 2 步 · 架构：仿射投影层

把网络输出写成 基线解 + 零空间残差： ŷ = A⁺x + (I−A⁺A)fθ(x)。 残差被投进 A 的零空间 → 在论文的无噪线性模型下，下采样回去恒等于 x，一致性是结构强制的，不靠损失函数软性鼓励。

第 3 步 · 化归：MAP = 最小化交叉熵

有了投影，约束优化变无约束，摊销 MAP 目标恰好等于最小化 H[qθ, pY]（模型输出分布 vs 真实图像分布）。 于是不再需要成对回归，问题变成生成式建模。

第 4 步 · 三种解法

AffGAN：修正 GAN 更新最小化反向 KL（差一个熵项的替代目标）——真实图像上视觉最好； AffDG：去噪器估计 ∇log pY 反传——偏糊； AffLL：PixelCNN+MCGSM 显式密度当先验——偏糊。

开场 30 秒背下来的话

"超分不是'从低清唯一还原高清'，而是在满足约束的候选里，挑一个最像真实图像的点。 本文用仿射投影锁住'满足约束'，用GAN推向'像真实图像'，中间用交叉熵把两件事接起来。"

③ 预备知识五张卡（零基础速成）

每张卡 = 核心结论 + 一句人话 + 自测。自测答不上来就点开看答案。

卡 1点估计与贝叶斯：损失函数决定"最优答案"落在哪

核心结论：后验 p(y|x) ∝ p(x|y)·p(y)（后验 ∝ 似然 × 先验）。 对同一条后验：平方损失 MSE 的贝叶斯最优解是均值；绝对损失 MAE 是中位数；0–1 损失（离散情形严格成立）是众数，即 MAP。

人话：分布像一座山——均值是"重心"，中位数是"中点"，众数是"峰顶"。超分多解时把解"平均"就会掉进两峰之间的空地，所以要爬峰顶。

自测：为什么 MSE 训练的超分网络输出会糊？
因为一张低清图对应多个合理高清解，MSE 的贝叶斯最优行为是输出这些解的后验加权平均；不同纹理逐像素平均后高频细节相互抵消 → 过度平滑。这不是网络不够大，而是目标函数本身决定的。

卡 2线性代数：零空间 · 伪逆 · 投影

核心结论：下采样是线性变换 x = Ay（带步长的卷积）。 零空间 = 被 A 压成 0 的方向，沿它移动，下采样结果不变； 伪逆 A⁺ 满足 AA⁺A = A，是"最稳的反操作"（实现为转置卷积，核需数值求解，A⁺ ≠ Aᵀ）； (I−A⁺A) 恰是投影到零空间的算子。

人话：A⁺x 给一个"低频肯定对"的底图，网络只负责往零空间里补高频细节——怎么补都不破坏与输入的一致性。

自测：为什么 ŷ = A⁺x + (I−A⁺A)fθ(x) 下采样回去一定等于 x？
两项分别看：A·A⁺x = x（在论文无噪线性模型、A⁺ 准确的前提下，A⁺x 本身就是一个合法解）；A·(I−A⁺A)f = (A−AA⁺A)f = (A−A)f = 0（用 AA⁺A=A）。相加即 Aŷ = x。

卡 3信息论：熵 / 交叉熵 / KL，一条恒等式

H[q, p] = H[q] + KL[q‖p] 读法：q 和 p 的交叉熵 = q 自身的熵 + q 到 p 的 KL 散度

人话（编码视角）：H[q] = 用 q 自己的最优编码表，平均最少花几个 bit；H[q,p] = 数据来自 q 却错用 p 的编码表，平均花几个 bit；KL = 因错配多花的那截。

为什么本文必须区分它们：MLE 里第一个分布是固定的真实分布，最小化交叉熵 ⟺ 最小化 KL；但本文第一个分布是随 θ 变的 qθ，H[qθ] 不固定 → 交叉熵目标（纯 MAP）与 GAN 最小化的 KL 不等价，恰好差一个"熵奖励" H[qθ]。

自测：GAN 的"熵奖励"是好事还是坏事？论文怎么说？
论文说是有益的副作用：相比纯 MAP（会把一切压向众数），KL 目标额外偏好更高熵、整体上更多样的解；§5.6 还指出熵项让 AffGAN 对坐标变换更有韧性。注意"缓解 mode collapse"是引申推论，论文原文没有这个说法——被追问时要说清这层区别。

卡 4GAN：本文用法与两处改造

核心结论：固定 G 时最优判别器 D* = pY / (pY + qG)。 本文两处改造：① 生成器输入不是噪声 z 而是低清图 x（带投影的 SR 网络就是生成器）；② 用修正的生成器更新规则（式 10 第二行），可证等价于最小化反向 KL[qθ‖pY]（附录 A，最早见于 Huszár 2016 博客）。

实例噪声（附录 C）：GAN 不稳的病根 = pY 与 qθ 都是集中、支撑几乎不重叠的分布 → 近似最优判别器一大堆、梯度乱。 解法：给真、假样本同时加高斯噪声（σ 退火到 0），两分布变胖重叠 → 最优 D 唯一、梯度稳；且与单边标签平滑不同，不给最优判别器引入偏差。

自测：与 SRGAN 最本质的三点区别？
① 一致性：SRGAN 无显式 LR 一致性约束（靠成对内容损失间接保证），AffGAN 在无噪线性模型下由仿射投影硬保证；② 数据：SRGAN 必须成对 LR–HR，AffGAN 目标原则上只需 pX、pY 的样本和已知 A；③ 理论：SRGAN 是"对抗+感知损失提质"的经验组合，AffGAN 从摊销 MAP 出发推导出"为什么可以用生成式目标"。

卡 5去噪 = 学对数密度的梯度（score）

( f*(ỹ) − ỹ ) / σ² ≈ ∇ỹ log pY(ỹ) 读法：去噪输出减带噪输入、除以 sigma 平方，约等于对数密度的梯度（仅小高斯噪声极限下成立）

人话：练好一个去噪器，它的"去噪方向"就近似指向"数据密度上升最快的方向"——不用写出先验密度，也能知道往哪儿走更像真图。AffDG 就是把这个梯度按链式法则反传去训练超分网络（论文自述：首次把去噪器输出显式反传训练另一个网络）。

自测：这和今天的扩散模型是什么关系？
思想同源（score matching，可追溯到 Vincent 2011、Alain & Bengio 2014），扩散模型同样靠"去噪 ≈ 估计 score"。但训练流程不同：这里是用冻结去噪器的梯度做引导训练前馈网络，不是完整的多步加噪-去噪生成过程。回答时点到"同源但非同一方法"即可。

④ 六个关键公式（含"照着念"读法）

讲公式的铁律：先中文、后符号。每条下面灰字就是你嘴里要说的话。

式 (4) · MAP 的贝叶斯分解

log p(y|x) = log p(x|y) + log pY(y) − log pX(x) "后验 = 似然 + 图像先验 − 一个与 y 无关的常数（可忽略）"。剩两件麻烦事：似然要求 Ay=x 严格成立，先验难建模。

式 (7) · 仿射投影层（核心创新）

gθ(x) = ΠAx fθ(x) = (I − A⁺A) fθ(x) + A⁺x "输出 = 零空间里的残差 + 直接上采样的基线解"。残差下采样恒为 0 → 一致性由结构保证。

式 (9) · 魔法一步：MAP 化归交叉熵

argmaxθ Ex log pY(ΠAx fθ(x)) = argminθ H[qθ, pY] "让输出在真实先验下平均对数概率最大 ⟺ 最小化输出分布与真实分布的交叉熵"。从此不需要成对数据 → 变成生成式建模。

式 (10) · 修正 GAN 目标

L(G;D) = −Ez log [ D(G(z)) / (1 − D(G(z))) ] "生成器不用原版损失，改用 log D 比 1−D"。附录 A 证明：D 最优时这等价于最小化反向 KL[qθ‖pY]。

熵恒等式 · 交叉熵 vs KL 的差

H[qθ, pY] − KL[qθ‖pY] = H[qθ] "我们想要的（交叉熵/纯 MAP）− GAN 实际最小化的（KL）= q theta 自己的熵"。GAN 额外奖励多样性，是有益副作用。

式 (12) · 去噪 ≈ score（AffDG 的根基）

( f*σ(y+σε) − y ) / σ² ≈ ∂/∂y log pY(y) "贝叶斯最优去噪器的残差除以 sigma 平方，近似对数密度梯度"——仅小噪声极限下成立，需退火。

⑥ Q&A 预演：最可能被问的 8 个问题

点开看参考答法。答不上的问题，诚实说"论文没做这个实验，我的理解是……"永远比编一个强。

为什么 GAN 最小化的是 KL 而不是交叉熵？差那个熵项要紧吗？
修正更新规则在最优判别器下对应反向 KL（附录 A）；与纯 MAP 交叉熵目标恰差 H[qθ]。方向上这是好事：纯 MAP 会把所有概率压向众数，熵项奖励整体多样性；瑞士卷实验里 AffGAN 交叉熵 4.10 也确实接近 MAP 的 3.15，说明实践中偏差不大。
A 不知道怎么办？真实退化不是高斯下采样呢？
诚实回答：本文明确假设无噪线性模型 x=Ay 且 A 已知（实验中 A 固定为高斯核）。A 未知或有噪声时投影不再严格成立——这是方法边界。可以补一句：软约束版（SoftGAN）不需要精确 A，但一致性和效果都更差，见 Table 1/2。
伪逆 A⁺ 是怎么得到的？是 Aᵀ 吗？
不是。A⁺ ≠ Aᵀ。论文把 A⁺ 实现为转置/亚像素卷积，核参数通过数值优化求得：最小化 ‖Ay−ABAy‖² + ‖Bx−BABx‖²（附录 B，蒙特卡洛 + SGD），收敛残差 10⁻¹²–10⁻⁸。Figure 2 还显示：A⁺ 必须初始化正确，否则性能受损。
MAP 真是我们想要的吗？作者自己不是也批评了？
对，§5.6 两点自我批评：① 众数依赖表示（换坐标/色彩空间答案会变，但 AffGAN 因熵项对此更有韧性）；② 高维下众数非典型（肥皂泡：典型样本范数 ≈ √d，众数范数 0）。作者给的出路是附录 F：给生成器加噪声 z，随机版 AffGAN ≈ 摊销变分推断，从近似后验采样而非只取众数。
和 SRGAN 比到底谁好？
论文没有和 SRGAN 做直接数值对比，比的是不同损失下的同架构模型。区别在定位：SRGAN 是"对抗+感知损失"的工程组合、需成对数据；AffGAN 给出概率解释（摊销 MAP）+ 结构一致性保证，原则上不需成对数据。两者互补，论文也说自己为 Ledig 等的 GAN-SR 路线"提供了理论动机"。
实例噪声和扩散模型的加噪是一回事吗？
直觉相通（加噪拓宽支撑、让学习信号更平滑，且都退火），但目的不同：实例噪声是为了让判别器的最优解唯一、稳定 GAN 训练；扩散模型的噪声调度定义了生成过程本身。同源不同用。
确定性模型怎么谈"多样性"？同一个 x 不是只有一个输出吗？
对——熵奖励作用于边缘输出分布 qθ（跨不同 x 的整体多样性），单个 x 仍只有一个确定输出。想要"同一 x 多个解"要用附录 F 的随机版（输入加 z）。这个区分要主动说清，不然容易被抓。
为什么 AffDG / AffLL 理论漂亮却输了？
都收敛但糊（附录 E）。AffDG：score 估计只在小噪声极限准，大噪声方向粗、小噪声覆盖窄，退火难调、易发散；AffLL：准的密度模型在数据流形附近太"陡"，训练初期几乎没有可用梯度；且密度模型（−4.10 bits/dim）仍不够准到能当可靠打分器。GAN 的判别器信号相对"平滑可用"，所以赢在工程可行性。

⑦ 红线清单：讲台上绝不能说错的话

左边是错误说法（✗），加粗绿字是正确版本。这些都是审阅时校准过的表述。

  • "下采样核 A 是学出来的" → A 是固定的 9×9 高斯核；只有 A⁺ 是数值优化求得的（Figure 2d 两个核别说反）。
  • "GAN 最小化的就是我们的 MAP 目标" → GAN 最小化反向 KL，与交叉熵 MAP 目标恰差一个熵项 H[qθ]，是替代目标。
  • "熵奖励缓解了 mode collapse（论文说的）" → 论文原话是"偏好更高熵、更多样的解"；mode collapse 是我们的引申。
  • "AffGAN 视觉效果最好，已被人类评测证实" → 本文没有人类偏好实验；说"本文实验中视觉上最锐利/最佳"。
  • "输出保证等于后验众数" → 是"近似 MAP 解"；且一致性保证的前提是无噪线性模型 + A⁺ 准确。
  • "AffDG 的去噪器全程一套参数冻住" → 不被超分损失更新，但训练中按退火计划不断换用更低噪声水平预训练的 DAE 参数。
  • "附录 F 配图见 Figure 8" → Figure 8 是 ImageNet 的额外结果图；附录 F 没有配图。
  • "MLE = 无信息先验下的 MAP" → 说"均匀（平坦）先验下的 MAP"更严谨。
  • "Figure 1 里的彩色点是数据点" → 是各方法的超分输出；且图 a 蓝点=MAP、图 b 蓝点=AffGAN，别混。
