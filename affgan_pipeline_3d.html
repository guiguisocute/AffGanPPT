现场求伪逆：SGD 训练 B ≈ A⁺

B = (0.000, 0.000)　AB = 0.000（目标 1）
损失 ≈ –　迭代 0

从零初始化的梯度下降有“隐式偏好”：停在最小范数右逆 (1,1) = 真伪逆。随机初始化则常停在别的右逆——b₁+b₂=2 的整条线都能把损失压到 0，Bx 依旧合法，但不再是“最保守”的基线。这正是论文强调“必须初始化成正确伪逆、训练中冻结”的原因。

推理四拍 · 一次前向出图

① 一张新低清图 x 进来（拖滑块换一张）
② A⁺x 基线放大：低频正确、细节模糊（黄球）
③ f_θ(x) 前向一次（白球）→ (I−A⁺A) 高通只留细节（紫箭头）
④ 相加输出 ŷ（蓝球）｜一致性 |Aŷ−x| = –

零优化迭代、零采样：一次前向就是近似 MAP 解——“摊销”在这里兑现。白球是网络真实的前向输出：它偏离约束线的对角分量来自随机初始化、训练从未改变过它（梯度穿过投影层被高通滤波，传不到这个方向）；投影一步把它精确扔掉。

PRSR 对比：采样 vs 众数

已从后验采样 0 张（洋红点，落满每一个峰）

PRSR 逐像素自回归地从后验采样：慢，但多样；AffGAN 一步输出众数（蓝球）：快，但唯一。同一个贝叶斯问题的两种取舍。
